# What is React:
1. react is JavaScript library for building web and mobile user interfaces.
2. react is declarative which requires to describe what should like ui and react handle all rest of operations not imperative.
3. react focus on only user interface not handle routing, fetching data from api.
4. react is powerful ecosystem which provides you other libraries to handle rest of your app like react-router-dom to handle routing and so on.
5. react focus on building your app in many layers and each layer do one task but its powerful which component can handle styling (css), html (jsx) and logic (js) inside component.
6. react focus on dividing ui in small pieces or chunks and separate each chunk in component.
